アイカツプラネット！ 第23話　夢の中へ

 Jun 13, 2021      アイカツプラネット 

プラネットプリンセスグランプリで優勝した舞桜のもとに仕事のオファーが殺到していて事務所もてんてこ舞いに。紗良さんまでかり出されています。

そこにやってきた舞桜

いずみんたち、てんてこ舞いだよぉ

みんなハナに興味津々なんだよ！

舞桜ちゃん、これから忙しくなるぞぉ！

ご褒美ステージ。内容は全部舞桜が決めていいみたいです。

学校にて栞と相談。

ごほうびステージ案…お菓子一杯のプール、風船で空を飛ぶ…

案の書かれたノートを敬斗に取り上げられます。早くステージを見たいから現実的な案を考えてくれないと困る…

楽しみにしてないこともない…ユリカさまな敬斗。

帰り道、妄想大好きなるりちゃんと、叶えられるかどうかは別として自由に想像するのが大事じゃない、という響子先輩。

るりちゃんも考えていることがあるようで、メルティ―ハウスのデザインを披露します。

ドラカツする響子先輩。アイカツにバンドを取り入れてみたい

栞は童話の執筆をしていて、いつか自分でも演じてみたいという夢

いずみマネージャーに呼び出しを受けた舞桜は3人を残して事務所へ向かいます。

まずは写真撮影から

空いた時間にファンレターへの返事

「アイドルになりたかった猫」　猫役は瀬川マネージャー

予定されていた明咲とのインタビューが遅れるようで、それを聞いた紗良さんがコーヒーをもってやってきます。いずみんは人使いが荒いから気を付けるように、と。

ドレスのデザインについて舞桜に相談。舞桜は右側の明るいデザインの方が好きみたい。

何に使うかは未定…まだ見ぬドレシアのためにドレスデザインを考えているようです。

想像が創造する。自由なイマジネーションが未来をクリエイトする。

そこにやってきた明咲。愛弓たちも同じようなことを言っていたらしく、ユニット名のお披露目会見の様子を2人に見せて上げます。

lovely marron

ね？ （…同じようなこと、言ってたかな？）

ご褒美ステージのことを尋ねる明咲。

何も決まってないってことはどんなことでもできるってことだね！　紗良さん！素敵！

メルティ―ハウスの新作、このドレス、るりちゃんがデザインしたんだ！

ドラカツ！

夢の国のおともだち…

わたしが笑顔になるのって、みんなのことを考えてる時かも

大好きなみんなと一緒のステージ、ご褒美ステージの内容が決まってきたみたいです。

スペシャルイマジネーションライブ。

司会はいずみマネージャー

アバター初披露の瀬川マネージャー

今日の内容は…「わたしの妄想がたーっぷり詰まったステージです！」

アイドル大集合！

ドレスデザイナーがアイドルでも良いよね！おもいっきり歌っちゃう！

キューピット推しだった…

「わたしの心の奥にあるアツいものはありがとうの気持ち！」
「もっともっと階段を駆け上がりたい！」
「どんどん失敗して宝物を集めちゃう！」
「音楽と一体になってビートを感じる！」
「親愛なるドレシアとともに輝く！」
「いつでもスイートでラブリーに♡」
「高く高く宇宙の彼方まで！」
「自分をさらけ出して感じるままに！」

奇跡を起こそう！みんなで！

全員PRドレスのステージ！ Bloomy*スマイル！

みんなの笑顔！もっと咲かせて！

実写パートに！

カップル誕生

いずみマネージャーも良い表情！

まだまだ続くよ！わたしたちのアイカツ！

「未来はキラキラ輝いてる。どんな素敵な未来もきっと作れちゃう。」

8人ステージ、プラネットからの実写…最高！これに尽きます！

一人一人自分の夢に向かって進んでいく、最終回に相応しい内容でした。

それから紗良さんの細かい気遣いが随所に表れていたのも。紗良さんはまだ見ぬドレシアのためにドレスを作っていて、これから新しいタイプのドレシアが出てくることもあるのかな。

来週は特別編、おはなしチャンネルは総集編という感じになるのかな。そのあとはどうなるんだろ？

…今回のステージもプラネットと現実世界の境界が曖昧になってるから…というSF的展開があってもいいのでは。
