tmuxコマンド

 Aug 21, 2020      Note 
tmux内での操作 bashでの操作 コマンドの説明
ヘルプコマンドとコマンドライン入力
C-b ? Help
C-b / キーの説明を表示
C-b : コマンド入力プロンプトの表示
アタッチとデタッチ
C-b d セッションを離れ、bashに戻る
tmux attach セッションに戻る
セッション操作
C-b : list-session tmux ls 現在開いるすべてのセッションを確認
C-b : kill-server 全てのセッションをキルする
C-b : kill-session カレントセッションをキルする
C-b s セッションをリスト表示
C-b $ セッション名を変更
ウインドウ操作
C-b c 新しいウインドウを作る
C-b % windowを左右に分割
C-b " windowを上下に分割
C-b 0, 1, 2 … カレントウインドウをwindow 0, 1, 2 … にする
C-b n, p 次/前のウインドウに移動
C-b w ウインドウをリスト表示
C-b & ウインドウをキルする
C-b , ウインドウ名を変更
ペイン操作
C-b UP, DOWN, RIGHT, LEFT 上下左右のペインに移動
C-b q 0, 1, 2 … ペイン番号を表示して、定した番号のペインをアクティブにする
C-b x アクティブペインをキルする
C-b o 次のペインに移動
コピー・貼り付け
C-b [ コピーモードに切り替え。リージョン選択後,C-wでコピー
C-b ] 最新のコピー内容を貼りつける
C-b = ペーストバッファーモードを起動
その他
C-b :set -g mouse on マウス操作を有効にする

.tmux.confに set -g mouse on と書いてホームディレクトリに保存すると起動後から有効になる。
